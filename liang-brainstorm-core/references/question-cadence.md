# Question Cadence

Shared question format for all liang-brainstorm-* skills.

## One Question at a Time

Ask **one main question at a time**. Never batch multiple questions into a single turn.

## Question Format

Each main question must include:

- **4 clear options:** `A`, `B`, `C`, `D`
- **`Recommended:`** one option, with reason
- **`Tradeoff:`** the main downside of the recommendation
- **`Confidence:`** qualitative only — `Low`, `Medium`, `Medium-high`, `High`
- **`Manual:`** invite the user to answer in their own words

## Rendering

Use a lettered-list format. Never use markdown tables for options:

```text
A. Label — description of the option.
B. Label — description of the option.
C. Label — description of the option.
D. Label — description of the option.

Recommended:
A — reason.

Tradeoff:
The main downside.

Confidence:
Medium-high.

Manual:
Describe in your own words.
```
