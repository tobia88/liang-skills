# Terminology

Shared terminology for all liang-brainstorm-* skills.

## Formal Terms

Use neutral formal wording for artifact names:

- `Strategy Report`
- `Brainstorm Report`
- `Decision Memo`
- `Project Brief`
- `Planning Notes`

## JRPG Labels

Use JRPG labels sparingly and functionally:

- `Main Quest` = goal/project brief
- `Victory Conditions` = success criteria
- `Boss Board` = risks
- `Fog of War` = unresolved questions/unknowns
- `Side Quests` = deferred ideas/non-core options
- `Quest Log` = Q&A reasoning trail
- `Save Point` = chat-only recap, not file saving

## Codex Warning

Avoid making `Codex` the official artifact name because it can be confused
with GPT/OpenAI Codex. If used, use it only as occasional flavor.
