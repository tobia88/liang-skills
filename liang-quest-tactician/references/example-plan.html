Create the logging configuration file with structured JSON output format.

Implementation Instructions
Create src/config/logging.json with the following content: { "format": "structured", "level": "info", "outputs": ["stdout"], "timestamp": true, "fields": { "service": "my-app", "version": "2.0" } } This replaces the implicit logging defaults from v1. The "structured" format enables JSON log lines instead of plain text. Match the indentation style from the existing app-config.json (2-space indent).
